# :question: Frequently asked questions

## :computer: How do I use NitroWin in a VM?

1. Download the [latest version of the answer file wrapped in an ISO file](https://github.com/nitrowinproject/NitroWin/releases/latest/download/unattend.iso).

2. (Optional) Modify the ISO file, so it includes your config file.

3. Add a secondary CD drive and insert the downloaded/modified ISO file.

4. Install Windows as usual.
